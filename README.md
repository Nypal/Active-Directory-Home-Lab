> ## Role Alignment
>
> This lab aligns with IT Support Engineer, Systems Administration, and OpsTech IT responsibilities, including user management, Windows Server administration, DNS, Group Policy, access control, troubleshooting, and root cause analysis.
>
> ---
>
> ## Objective
>
> To build a fully functional Active Directory home lab environment that simulates a real enterprise IT infrastructure. This project demonstrates hands-on experience with domain controller setup, user and group management, Group Policy configuration, DNS administration, and IT troubleshooting — all core competencies required for IT Support Engineer and Systems Administrator roles.
>
